# Getting Started with Astro 5.0

Astro 5.0 introduces a new way to build content-driven websites. With **Content Layers**, you can fetch data from any source and use it in your Astro project.

## Key Features

1. **Content Layers**: A flexible way to manage content.
2. **Server Islands**: Dynamic components on the server.
3. **Enhanced Performance**: Faster build times.
